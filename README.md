### Installation et Lancement du Serveur Local

1. Allez dans le dossier du serveur :


2. Installez les dépendances :


3. Lancez le serveur local :
- Avec nodemon (pour une mise à jour du serveur sans redémarrage) :
  ```
  nodemon start
  ```

  ```
  npm start
  ```
